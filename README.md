# Immersive Industry

Minecraft Mod for 1.16.5 (Forge) and 1.20.1 (Forge/NeoForge)

## Content

We added new Multiblocks:

- Crucible
- Steam Turbine
- Industrial Electrolyzer
- Car Kiln
- Rotary Kiln

We added new Machines:

- Electrolyzer

## Note

This is a Library Mod for Modpack Creators. It does not bundle with Recipes.
